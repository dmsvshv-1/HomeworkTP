# HomeworkTP

Учебный проект на Python. Написал несколько классов для работы с рецептами и списком покупок.

## Что тут есть

**Ingredient** — ингредиент. У него есть название, количество и единица измерения (граммы, штуки, миллилитры и т.д.). Количество не может быть отрицательным.

**Recipe** — рецепт. Хранит название и список ингредиентов. Можно добавить ингредиент — если такой уже есть, количество просто суммируется. Ещё можно масштабировать рецепт, например умножить все ингредиенты на 2, если готовишь на большую компанию.

**ShoppingList** — список покупок. Добавляешь туда рецепты с количеством порций, он сам считает сколько чего нужно купить. Одинаковые ингредиенты из разных рецептов объединяются. Список можно получить отсортированным по алфавиту. Два списка покупок можно сложить через `+`.

**DietaryRecipe** — то же самое что Recipe, только с типом диеты (например, веганское, без глютена и т.д.).

## Установка

```bash
git clone <ваш-репозиторий>
cd <папка>
pip install -r requirements.txt
```

## Как запустить тесты

```bash
pytest
```
